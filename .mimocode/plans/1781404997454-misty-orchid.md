## Key Constraints

1. **Android**: Requires Android Studio + device/emulator for testing. Cannot verify Kotlin compilation in Rust-only CI.
2. **iOS**: Requires Xcode + macOS for testing. Cannot verify Swift compilation in Rust-only CI.
3. **UniFFI**: Generated bindings must match the UDL. Run `cargo run --bin gen_kotlin --features gen-bindings` after UDL changes.
4. **Permissions**: Android 13+ requires `NEARBY_WIFI_DEVICES` runtime permission. iOS requires `bluetooth-central`/`bluetooth-peripheral` background modes.
5. **API levels**: WiFi Aware requires API 26+. WiFi Direct requires API 14+. L2CAP CoC requires API 29+.
