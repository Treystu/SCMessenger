# SCMessenger Repository

## Overview
SCMessenger is a cross-platform messaging application built using modern web technologies and frameworks.

## Setup Instructions
1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-repo-url.git
   cd your-repo-directory
   ```

2. **Install dependencies:**
   - For Android:
     ```sh
     ./gradlew build
     ```
   - For iOS:
     ```sh
     pod install
     ```

3. **Run the application:**
   - For Android:
     ```sh
     ./gradlew run
     ```
   - For iOS:
     Open `SCMessenger.xcworkspace` in Xcode and run the project.
